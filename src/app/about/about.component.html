All About the Client Stack!

As one of my final projects at the CareerDevs Computer Science Academy, I decided to build a full stack web application that a hair stylist could use to organize their client information including name, contact, services, last and next appointments, and a calendar to display upcoming appointments. The inspiration for this project came from watching my girlfriend use a book to keep track of her client info and it didn't seem very practical, nor does she have any backup for that book in the case that something were to happen to it. Will I ever get her to use my application? Who knows! But I can say that I learned a ton about building a REST API with Java Spring, and a front end using Angular as I developed this application.

Technologies Used Angular Angular Material Angular-Calendar Java 8 Spring Maven MySql Front End Repo Back End Repo
